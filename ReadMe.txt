_________________ My Files Backup by Demston _________________

My Files Backup v.2

Простая программа для копирования и архивирования твоих файлов.
При копировании сохраняет структуру вложенных папок.
Можешь прописать множество исходных путей, содержимое которых 
в итоге окажется в твоей целевой папке (архиве), исходя из условий.

ВАЖНО! После пятой вложенной папки программа не анализирует 
даты создания и изменения дальнейшего содержимого.
